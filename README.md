# emulator
Simple 6502 CPU emulator.

Designed and modified from this project: https://github.com/davepoo/6502Emulator

## Specs
CPU
* 8 bit word size (bus size)
* 1 thread
* stack - 256 bytes
* 3 general purpose registers

RAM - 64Kb
